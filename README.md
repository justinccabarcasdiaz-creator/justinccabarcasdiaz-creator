## 📌 Descripción

Este proyecto es una interfaz web desarrollada **únicamente con HTML5 y CSS3**, sin dependencia de frameworks o librerías externas. El objetivo es demostrar el poder del CSS moderno aplicando técnicas avanzadas como **Grid Layout**, **Flexbox**, **Custom Properties**, animaciones y diseño responsivo.

> 💡 Ideal como base para proyectos web, portafolios o como referencia de buenas prácticas en frontend.

---

## ✨ Características

- ✅ **Semántica HTML5** — Estructura accesible y bien organizada con etiquetas semánticas
- ✅ **CSS3 Moderno** — Variables CSS, animaciones, transiciones y pseudoelementos
- ✅ **Flexbox & Grid** — Layouts fluidos y centrados con las herramientas CSS más potentes
- ✅ **Diseño Responsivo** — Adaptable a móvil, tablet y escritorio mediante Media Queries
- ✅ **Sin dependencias** — Cero librerías externas, cero JavaScript requerido
- ✅ **Alto rendimiento** — Código limpio, optimizado y de carga rápida
- ✅ **Accesibilidad** — Contraste adecuado, atributos `alt`, estructura de encabezados lógica

## 💻 Uso

Una vez abierto en el navegador, podrás explorar todas las secciones del proyecto. Para personalizar:

1. **Colores** — Modifica las variables en `:root` dentro de `styles.css`
```css
:root {
  --color-primary: #6c63ff;
  --color-secondary: #f5a623;
  --color-bg: #ffffff;
  --font-main: 'Your Font', sans-serif;
}
```

2. **Contenido** — Edita directamente el `index.html`

3. **Tipografías** — Cambia el `<link>` de Google Fonts en el `<head>`

| 💻 Tablet | `768px – 1023px` |
| 🖥️ Desktop | `1024px – 1279px` |
